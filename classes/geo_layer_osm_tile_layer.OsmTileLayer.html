Class OsmTileLayer

A base class for Layer subclasses. It provides some utility functions to Layer and also gives access to $iApi and $vApp globals. Mostly it exposes stub methods on LayerBase; this is because layer subclasses can be wildly different, so we don't have a pile of common things to put here. The stubs will help debugging as they will alert devs when they have not implemented something. The stubs also allow us to get intellisense / typescript happiness when dealing with common layer variables typed as LayerInstance.
